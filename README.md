# ITrizz for Valentine's Day

A simple static web project for a Valentine's Day preview.

## Setup

- Clone the repo or download the ZIP and open the project folder.

  ```bash
  git clone https://github.com/your-username/ITrizz-for-Valentine-s-Day.git
  cd ITrizz-for-Valentine-s-Day
  ```

- Open the project in your browser by double-clicking `index.html`, or run a simple local server for a better experience:
  - Using Python (works in PowerShell / Terminal):

    ```bash
    python -m http.server 8000
    # then open http://localhost:8000 in your browser
    ```

  - Or use the VS Code Live Server extension and open `index.html`.
